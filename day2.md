### 원격 저장소 연결 

```bash
git remote add origin https://github.com/jibook/TIL.git
```



### 원격 저장소 연결 여부 확인

```bash
git remote -v

origin  https://github.com/jibook/TIL.git (fetch)
origin  https://github.com/jibook/TIL.git (push)
```



### 원격 저장소에 보내기

```bash
git push origin master
```



### 버전 로그 보여줌(커밋 로그 보여줌)

```bash
git log
git log -p
git log --one line
```
